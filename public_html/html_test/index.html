バリアフリーマップ検索

適当に住所を打ち込んでエンターを押すと検索してくれます．あと左のメニューに項目が追加されます．

実装メモ

流れ

  1. キーワード・住所を入力
  2. Google Maps APIで住所に該当する地点に移動
  3. 対応する地点のGPSアドレスおよび，zoomレベルより，地図に表示される範囲を導出
  4. サーバにクエリを投げる (住所・キーワードを分離しておく)
  5. 周辺地域を優先的に表示 (周辺にない施設も与えられる．)

問題点

  • 「キーワード 住所」形式の検索だと住所が参照できない

    解決策

    1. キーワードを削って再検索
    2. GlocalSearch (Google AJAX Search API)を使う
    3. 住所入力欄とキーワード欄を分ける
